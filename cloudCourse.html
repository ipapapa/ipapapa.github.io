Cloud Computing by Ioannis Papapanagiotou

Overview

This course provides an introduction in the techniques and technologies used in cloud computing. It consists of independent and intensive hands on labs. The course emphasizes on architecture and the development of web services that can scale on cloud infrastructure. The students learn how to deploy an IaaS, develop applications against common PaaS vendors. Students or attendees learn how to focus on service decoupling and the principles of a cloud native enterprise. Below we show some of the modules offered in this course.

The cloud computing has been constantly evolving, and variations of it have been offered at Purdue University and North Carolina State University.

Instructor: Ioannis Papapanagiotou

Topics Covered

  • Introduction to Cloud Computing (standardization and automation, rapid elasticity, flexibility and pricing)

  • Architecture (Service and Delivery Models, Virtualization, Hypervisors)

  • IaaS and PaaS Cloud Services and Vendors (OpenStack, Amazon AWS, OpenStack, Google Cloud and Microsoft Azure etc)

  • Anatomy of the Cloud (Cloud tiers, Sharding, CAP Theorem, ACID/Base,Â  Web Services)

  • Replication, Consistency, 2/3 Phase Commits, and Paxos (Vector Clocks, Byzantine Failures, Chandy/Lamport)

  • PaaS Cloud Storage and Case Studies (The Amazon Dynamo/Cassandra, Google's BigTable etc)

  • Data Pipeline (Distributed Queues, Message Brokers, Pub/Sub, Service Decoupling, Exchange Types and Apache Kafka)

  • Micro Services and Containers (Control Groups, Namespaces, Docker, App management with Fig, Mesos Scheduling)

  • Network Virtualization (OpenStack Neutron, Software Defined Networks, OpenFlow)

  • Security (security integration model,threads, Economic Denial of Sustainability, threat mitigation, Google Case Study)

Hands on Labs

  • Introduction to VMWare vCenter based virtualization

  • OpenStack Deployment

  • Docker and Linux Containers

  • Architecting and Developing on AWS

  • Big Data on AWS

  • Developing client web services with Node.js, Go, Scala, Python etc.
